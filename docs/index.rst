Welcome to pyHSPF's documentation!
====================================

pyHSPF is a Python package for downloading and running HSPF (Hydrological
Simulation Program – Fortran) models.  It provides utilities for reading and
writing UCI configuration files, processing binary output (HBN), managing
Watershed Data Management (WDM) files, and building model workflows.

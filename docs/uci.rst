UCI Module
==========

The ``hspf.uci`` module provides a Python interface for reading, manipulating,
and writing HSPF UCI (User Control Input) files.  A UCI file is the primary
configuration for an HSPF simulation, describing the model's operations,
parameters, external data sources, network connectivity, and global settings.
